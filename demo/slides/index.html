What is MCP?

The Model Context Protocol (MCP) is an open standard that connects AI assistants to systems directly.

  • Think "USB-C for AI Applications"
  • Standardized way for AIs to access Data (files, databases) and Tools (functions, APIs).
  • Decouples the Model from the Context.

The Context Economy

The Cost of Context

  • Every connected server adds system prompts and tool definitions.
  • Too Many Servers = Slower inference & higher cost.
  • Solution: Toggle servers On/Off instantly based on task.
